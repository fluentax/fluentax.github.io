Network requirements

TLS​

TLS protocol version​

The minimum supported version is TLS 1.2. For enhanced security and performance, we recommend using TLS 1.3.

Certificate Authorities​

Fluentax uses Cloudflare as its CDN and security provider, so the Certificate Authority (CA) may be one of those listed by Cloudflare. You can find the full list of possible CAs here: Cloudflare Certificate Authorities.

If your system does not trust these CAs, please update your trust store to include the necessary root certificates.

caution

The root Certificate Authority may change in the future. Keeping your system up to date is the best way to maintain compatibility.

IP addresses​

If you need to allowlist Fluentax IP addresses, please refer to Cloudflare’s published IP ranges.

caution

Cloudflare’s IP ranges may change. We recommend subscribing to updates or monitoring IP list changes to avoid service disruption.

Firewall and proxy settings​

Ensure your firewall or proxy allows outbound HTTPS traffic (port 443) to Cloudflare IP ranges. Restrictive firewall policies may block necessary traffic and cause integration failures.
