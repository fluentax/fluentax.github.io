Terminology

Quote Type#

Exchange rates can be quoted in two ways, direct and indirect. The quotation type is dependent on the bank.

Direct Quote#

The cost of one unit of foreign currency is given in units of local currency.

Example:

{    "bankId": "NOCB",    "baseCurrency": "NOK",    "quoteType": "Direct",    "rates": {        "2020-09-24": {            "CHF": {                "rate": 1030.66,                "unitMultiplier": 2            }        }    }}

Interpretation: 100 CHF = 1030.66 NOK or 1 CHF = 10.3066 NOK

Indirect Quote#

The cost of one unit of local currency is given in units of foreign currency.

Example:

{    "bankId": "EUECB",    "baseCurrency": "EUR",    "quoteType": "Indirect",    "rates": {        "2020-09-24": {            "ARS": {                "rate": 88.0944            }        }    }}

Interpretation: 1 EUR = 88.0944 ARS

Note

Quote type is only available for JSON and XML response formats. For tabular formats (CSV, XLSX, SAP Flat File) the columns From Currency and To Currency represent the quotation.

Unit Multiplier#

Some banks specify certain rates in 10, 100, 1000, etc. units of the foreign currency. This is usually applied to currencies where the observed rate is so small that it couldn't be represented accurately using a fixed number of fractional digits without losing precision.

Fluentax FX API uses the SDMX Unit Multiplier notation to preserve the original precision and representation of these rates.

Unit Multiplier

Unit Multiplier is an exponent in base 10 specified so that multiplying the observation numeric values by 10^Unit Multiplier gives a value expressed in the unit of measure.

Examples (direct quote):

Unit Multiplier valueUnitDescriptionForeign CurrencyBank CurrencyRateInterpretation
<omitted> or <null> or 01UnitsEURHUF359.301 EUR = 359.30 HUF
110TensZARBGN1.060910 ZAR = 1.0609 BGN or 1 ZAR = 0.10609 BGN
2100HundredsVESCHF0.00005100 VES = 0.00005 CHF or 1 VES = 0.0000005 CHF
Tip

When unitMultiplier is specified, you can divide the rate by the ten raised to the power of unitMultiplier to get the actual rate.

C# example:

var actualRate = rate / (decimal)Math.Pow(10, unitMultiplier ?? 0);
Note

Unit Multiplier notation is only used for JSON and XML response formats. Tabular formats (CSV, XLSX, SAP Flat File) use a Unit column instead specifying the exact units (1, 10, 100, etc.).

Price Types#

There are three main price types: mid, bid and ask. Fluentax FX API uses mid rates by default. For certain banks, bid and ask prices will be made available in the future.

Mid (middle) price#

The average of the bid price and the ask price.

Bid (buying) price#

Bid or buying price is the price used by the foreign exchange bank to buy foreign currency from the customer.

Ask (selling) price#

Ask or selling price refers to the exchange rate used by the bank to sell foreign exchange to customers.
